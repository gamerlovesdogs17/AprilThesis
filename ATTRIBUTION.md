# Attribution

The canonical detailed record is `apps/web/public/assets/assets-manifest.json`.

## Natural Earth

Exterior geographic context derives from Natural Earth's public-domain 1:110m administrative countries dataset: https://www.naturalearthdata.com/about/terms-of-use/

## Historical portraits

Thirteen local portraits come from individually reviewed Wikimedia Commons file pages. Twelve are marked public domain and the Trotsky Rijksmuseum image is CC0 1.0. Names, dates, creators, source URLs, and crop history are recorded in `apps/web/public/assets/portraits/sources.json` and `docs/ASSET_SOURCES.md`. Commons page text is not redistributed.

## Project-created media

The congress-hall image is an original project-directed generated artistic reconstruction and is labeled as such. All distributed WAV files are original deterministic synthesis generated by the repository script. Strategic geometry, UI SVG, fallback silhouettes, the modern interface insignia, charts, newspapers, patterns, and animation are code-native project work.

The interface insignia is not a claimed historical Workers' Opposition emblem.
