# Asset Sources

No external media ships in the current build.

- Map geometry: original project SVG paths; no traced historical map.
- Political patterns and interface: original CSS/SVG.
- Intro graphics: browser-rendered original SVG/CSS.
- Wind ambience: generated at runtime with Web Audio noise; not a recording.
- Fonts: system fallbacks only; no bundled font files.

The manifest and validator require source, creator, license, and verification metadata before future files enter `apps/web/public`.
