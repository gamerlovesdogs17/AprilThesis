# Institutions

Eight implemented institutions affect the campaign: Politburo, Central Committee, Party Secretariat, Sovnarkom, All-Russian Central Council of Trade Unions, Cheka, local soviets, and factory-committee networks.

State tracks include player influence, contacts, security penetration, bureaucratization, and corruption. Content records formal/informal authority and leadership. The UI exposes the gap between formal power and player access.

The Secretariat description explicitly prevents anachronistic treatment of Stalin as 1921 General Secretary. The Cheka name remains in use throughout this chapter; GPU terminology is reserved for 1922 expansion.
