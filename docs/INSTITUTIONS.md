# Institutions

Eight implemented institutions affect the campaign: Politburo, Central Committee, Party Secretariat, Sovnarkom, All-Russian Central Council of Trade Unions, Cheka, local soviets, and factory-committee networks.

State tracks include player influence, contacts, security penetration, bureaucratization, corruption, attitude, autonomy, agenda, current business, and last action. Content records formal/informal authority and leadership. Institutions pursue visible autonomous monthly business, while player actions can build contacts, submit memoranda, or cultivate individual relationships through a limited political-action economy. The UI exposes the gap between formal power and player access.

The Secretariat description explicitly prevents anachronistic treatment of Stalin as 1921 General Secretary. The Cheka name remains in use throughout this chapter; GPU terminology is reserved for 1922 expansion.
